# Ideas

> Actionable items extracted from research. Candidates for becoming issues, ADRs, or implementation phases.

When the researcher marks an item `status: actionable` AND it has substantial "Adoptable patterns" filled in, the librarian flags it here.

To promote an idea:
1. Open a GitHub issue referencing the research file
2. Update the research file's `status: actioned` and add the issue link
3. The librarian will move the file from active rotation to acknowledged-tracking
